Comment utiliser GradeFlow ?

GradeFlow est conçu pour vous simplifier la gestion des barèmes et l'évaluation des élèves. Voici comment procéder :

1. Créez un nouveau barème

Cliquez sur "Créer un nouveau barème". Vous pourrez y ajouter des questions, définir un nombre de points pour chacune et personnaliser votre barème.

2. Chargez un barème existant

Si vous avez déjà un barème sauvegardé au format JSON, utilisez l'option "Charger un barème". Cela vous permettra de le réutiliser et de gagner du temps.

3. Évaluez vos élèves

Une fois le barème chargé, commencez à évaluer vos élèves. Saisissez les points obtenus pour chaque question et ajoutez, si nécessaire, des commentaires personnalisés.

4. Téléchargez les résultats

Une fois l'évaluation terminée, vous pouvez télécharger un fichier JSON contenant les résultats ou générer une page HTML récapitulant les performances de chaque élève.
